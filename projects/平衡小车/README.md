# 平衡小车

#### 介绍
这是达妙科技开源平衡小车仓库 QQ交流群：680048275 。

平衡小车硬件方案是达妙STM32H7开发板+两个达妙3510电机+蓝牙模块。

控制模型都是一阶倒立摆，控制方法用的是lqr。

如果该项目对您有帮助，可以点点右上角的star。
